🔗 Połączenie z GitHub

⚠️ Nie skonfigurowano
Wymagany scope: repo (odczyt + zapis)
Dodawany do powiadomień Telegram przy nowych kuponach.

📖 Jak uruchomić

1. Utwórz Personal Access Token
GitHub → Settings → Developer settings → Personal access tokens → Fine-grained lub Classic.
Scope: repo (pełny dostęp do prywatnych repozytoriów).

2. Wgraj webapp do repozytorium
Umieść pliki w folderze webapp/ w repozytorium.
Włącz GitHub Pages: Settings → Pages → Source: main /webapp.

3. Dodaj GitHub Secret WEBAPP_URL
W GitHub Secrets dodaj WEBAPP_URL z adresem tej strony.
Kupony wysyłane przez Telegram będą zawierały link do panelu.

4. Auto-resolve
Dodaj plik .github/workflows/auto_resolve.yml z folderu fix/.
Kupony będą automatycznie rozliczane co 6 godzin i po każdym daily_fetch.

5. Dane odczytywane/zapisywane
data/results/coupons_history.json — historia kuponów
data/results/finance.json — Twoje transakcje

⚠️ Token jest przechowywany w localStorage przeglądarki.
Nie otwieraj panelu na współdzielonych komputerach.
